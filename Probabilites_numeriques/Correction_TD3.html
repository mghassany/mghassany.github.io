Simulation de variables alatoires par inversion de la fonction de rpartition

Pour simuler (gnrer) une variable alatoire \(X\) de loi de rpartition \(F\) dont on sait calculer linverse gnralise \(F^{-1}\), on peut simuler une variable alatoire de loi uniforme \(U\) sur lintervalle \([0,1]\) et poser \(X=F^{-1}(U)\).

Exercice 1

  • Densit: \(f(x)=(1/2) cos(x) \times 1_{\{-\pi/2 \leq x \leq \pi/2\}}\)
  • Fonction de rpartition rciproque: \(F^{-1}(x)=arcsin(2x-1)\)
# cration de la fonction rcos qui va simuler N valeurs alatoires suivant la densit f(x) ci dessus.
rcos=function(N)
{
  u=runif(N,0,1)
  return(asin(2*u-1))
}

Vrification de la simulation par un histogramme:

par(mfrow=c(1,2))
# densit thorique
curve(1/2 * (cos(x)), from=-pi/2, to = + pi/2, main="courbe de la densit", lwd=3, col="red")
hist(rcos(10000), probability = T, col="red")

Remarque: il est utile aussi de superposer les deux figures:

Simulation de variables alatoires par la mthode du rejet

Rappels : Principe de la mthode du rejet Pour simuler (ou gnrer) une variable alatoire de densit \(f\) borne par \(M\) et support compact sur \([a,b]\), on simule des variables alatoires uniformes sur \([a,b]\times[0,M]\) (qui reprsente le pav contenant le graphe de \(f\)), jusqu ce que lun des points tirs appartiennent lhypographe de \(f\). On prend alors pour ralisation de la loi \(f\) labscisse de ce point.

Exercice 5

La loi de Wigner a une densit \(f(x)=\sqrt{4-x^2}/2\pi\) sur \([-2,2]\). On va dabord simuler des valeurs suivants cette loi. On va faire cette simulation manuellement (au contraire dautres lois dj prdifinies sur R, comme la fonction rnorm() qui permet de simuler des valeurs suivants la loi Normale).

La courbe de la densit est de la forme:

curve(sqrt(4-x^2)/(2*pi),from=-2,to=2)

Lide pour simuler un nombre fixe des valeurs suivant cette loi est de gnrer un des points sur le rectangle de la figures ci dessus, si le point est au dessous de demi cercle on le garde, sinon on le rejette.

Pour simuler un point on gnre son abscisse et son ordonn selon la loi uniforme. Le point est en dessous du demi cerle si son ordonn est infrieur \(\sqrt{4-abscisse^2}/2\pi\)

N<-10000
 # le vecteur contenant les 10000 valeurs simuls suivant la loi de Wigner
rWigner <- function(N){
U<-rep(NA, N)
for (k in (1:N)) 
{ # on gnre un point
  x<-runif(1,-2,2) # l'abscise selon la loi unifome entre -2 et 2   
  y<-runif(1,0,0.4) # l'ordonn selon la loi uniforme entre 0 et 0.4 (un peu en dessus du cercle)    
  while (y>sqrt(4-x^2)/(2*pi)) # si le point gnr est  l'extrieur de demi cercle on re gnre un autre point jusqu' que le point gnr est en dessous.
    {   x<-runif(1,-2,2)    
        y<-runif(1,0,2)
    }  
  U[k]<-x # on garde les abscisses
} # A ce moment, U contient N v.a. distribues selon la loi de Wigner. 
return(U)
}

Vrification de la simulation par un histogramme:

par(mfrow=c(1,2))
# densit thorique
curve(sqrt(4-x^2)/(2*pi),from=-2,to=2, main="courbe de la densit de Wigner", lwd=3, col="red")
hist(rWigner(10000), probability = T, col="red")

Extra 1: Simulation de la loi normale centre rduite

La fonction rciproque de la fonction de rpartition de la loi normale centre rduite nest pas calculable analytiquement. Une faon pour simuler cette loi est dutiliser la mthode de rejet.

La courbe de la loi normale centre rduite est toujours en dessous de la courbe de la loi exponentielle de paramtre 1:

curve(exp(-x), from=0,to=7, col = "blue", lwd=3, ylab="")
curve(exp(-0.5*x^2)/sqrt(2*pi), add=T, col="red",lwd=3,ylab="")
legend("topright" , legend=c("Exponentielle", "Normale centre rduite (sur R+)"),
       col=c("blue", "red"), lty=c(1,1), lwd=c(3,3), cex=0.6)

Lide est de simuler des points qui sont en dessous de la densit de la loi exponentielle. Donc les abscisses suivent la loi exponentielle, les ordonns suivent la loi uniforme sur \([0,1]\), et on multiplie les ordonns par la densit exponentielle \(e^{-x}\) pour avoir les points simuls en dessous de la densit exponentielle. On obtiens le suivant:

x = rexp(1000)
y=runif(1000)
plot(x,y*exp(-x), pch=21, bg="black")
curve(exp(-x), add=T, col = "blue", lwd=3)
curve(exp(-0.5*x^2)/sqrt(2*pi), add=T, col="red",lwd=3)

Ensuite, on rejette les points que sont entre les deux courbes:

x = rexp(1000)
y=runif(1000)
plot(x,y*exp(-x), pch=21, bg=ifelse(exp(-0.5*x^2)/sqrt(2*pi)<exp(-x)*y,"yellow","green"))
curve(exp(-x), add=T, col = "blue", lwd=3)
curve(exp(-0.5*x^2)/sqrt(2*pi), add=T, col="red",lwd=3)

Comme vous lavez remarqu, on garde les points en dessous de la densit normale, mais ceux ci ne couvrent que la partie \(\mathbb{R}^+\) de la loi normale centre rduite. Cest pour cela quon rajoute une condition altoire qui permet de garder loppos de labscisse avec une chance 1/2 : if (runif(1)<0.5){X[i]=-X[i]}.

#gnrer une loi normale  partir d'une loi expo, mthode du rejet
SimulN=function(N)
{
  X=rep(0,N)
  for (i in 1:N)
  {
    X[i]=rexp(1)
    Y=runif(1)
    while(1/sqrt(2*pi)*exp(-X[i]^2/2)<exp(-X[i])*Y)
    {
      X[i]=rexp(1)
      Y=runif(1)
    }
    if (runif(1)<0.5)
    {
      X[i]=-X[i]
    }
  }  
  return(X)
}

Simulons maintenant des valeurs un chantillon pour voir si on gnre bien une variable altoire normale centre rduite.

hist(SimulN(10000), probability = T, col="yellow")
curve(exp(-0.5*x^2)/sqrt(2*pi), add=T, col="red",lwd=3)

Simulation de loi gomtique

#Loi gomtrique, simulation par repetition de Bernouilli
rgeom=function(N,p)
{
  X=rep(0,N)
  for (i in 1:N)
  {
    X[i]=0
    T=runif(1)
    while(T>p)
    {
      X[i]=X[i]+1
      T=runif(1)
    }
  }
  return(X)
}

hist(rgeom(10000, 0.3), probability = T, col = "darkgreen")

Simulation de loi de Poisson

#methode du rejet, loi de Poisson (ex6)
rpoisson=function(N,L)
{
  X=rep(0,N)
  for (i in 1:N)
  {
    X[i]=rgeom(1,1-L)
    T=runif(1)
    while(L^X[i]*exp(-L)/factorial(X[i])<(L^X[i]*exp(-L)*T))
    {
      X[i]=rgeom(1,1-L)
      T=runif(1)
      
    }
  }
  return(X)
}

hist(rpoisson(10000,0.6), probability = T, col="darkblue")
